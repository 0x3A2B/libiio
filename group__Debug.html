__api ssize_t iio_device_read_raw ( const struct iio_device *  dev,
void *  dst,
size_t  len,
uint32_t *  mask,
size_t  words 
)

Read the raw stream from the given device.

Parameters
devA pointer to an iio_device structure
dstA pointer to the destination buffer where to write the stream
lenThe length of the destination buffer, in bytes
maskA pointer to a memory area where the channel mask will be stored
wordsThe number of 32-bit words composing the mask
Returns
On success, the number of bytes read
On error, a negative errno code is returned

NOTE: The device must be opened first (with iio_device_open).

The "words" param should correspond to (number of channels + 31) / 32. The area pointed by "mask" will be initialized like this: 

mask[chn.index / 32][chn.index % 32] = is_enabled

Note that the mask can change anytime between two calls, even if no channel of the specified device have been enabled or disabled in the meantime.
