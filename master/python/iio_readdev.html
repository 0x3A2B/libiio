iio_readdev¶

iio_readdev is part of the Libiio package, a library that has been developed to ease the development of software interfacing Linux Industrial I/O (IIO) devices.
This tool is written using the libiio Python bindings. It works in the same way as the base iio_readdev works. You can find more informations about it on this page.
iio_readdev.create_context(scan_for_context, arg_uri, arg_ip)[source]¶

Method for creating the corresponding context.

parameters:
scan_for_context: type=bool
Scan for available contexts and if only one is available use it.
arg_uri: type=string
The URI on which the program should look for a Context.
arg_ip: type=string
The IP on which the program should look for a Network Context.
returns: type:iio.Context
The resulted context.

Method for reading data from the buffer.
